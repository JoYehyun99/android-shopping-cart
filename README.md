# android-shopping-cart

## 기능 요구 사항

- 상품 목록
    - 상품 목록이 보여진다.
    - 상품을 클릭하면 상품 상세로 이동한다.
    - 상품 이름이 초과하면 말줄임표로 표시해야한다.
    - 장바구니에 담을 상품의 수를 선택할 수 있다.
      - `+` 버튼을 누르면 장바구니에 상품이 추가됨과 동시에 수량 선택 버튼이 노출된다.
      - 상품 목록 & 장바구니 상품의 상태가 동일하게 반영되어야 한다.

- 상품 상세
    - 상품 상세 정보를 알 수 있다.
    - 장바구니에 담을 수 있다.
    - 장바구니에 담을 상품의 수를 선택할 수 있다.
      - 수량 선택에 따라 가격이 변화해야 한다.


- 장바구니
    - 담은 상품 목록을 볼 수 있다.
    - x 버튼으로 삭제할 수 있다.
    - 상품의 수를 조절할 수 있다.
      - 수량 선택에 따라 가격이 변화해야 한다.



## 페어 규칙

- 서로 존중한다.
- 쉬는 시간을 적절하게 가진다.
- 혼자하지 않는다.
